## Interaction Flow

1. **View Calendar**
   - User clicks "Calendar View" in sidebar
   - Calendar initializes to current month
   - All dates with labs show status badges

2. **Click Date**
   - User clicks a date with labs
   - Side panel slides in from right
   - Shows all labs grouped by panel

3. **View Lab Details**
   - Each lab marker shows:
     - Test name
     - Value with units
     - Reference range
     - Status badge (Normal/High/Low)
   - Click any marker to view trend chart

4. **Filter Labs**
   - Select panel filter (CMP, CBC, etc.)
   - Or select specific marker
   - Calendar updates to show only filtered results

5. **Compare Dates**
   - Enable comparison mode
   - Click first date
   - Click second date
   - Side panel shows side-by-side comparison

6. **Navigate Time**
   - Use Prev/Next buttons
   - Or click "Today" to return to current month
   - Switch between Month/Week views
